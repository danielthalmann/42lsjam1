# Game jam 1.0 de 42 Lausanne

Marathon de création de jeux vidéo de 48h non-stop à l'école 42 Lausanne.

La Game Jam s'est déroulée du 19 au 21 avril 2024

<a href="https://42lsjam.ch/">https://42lsjam.ch/</a>
